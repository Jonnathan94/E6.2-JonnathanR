Qué és SASS

Sass significa “Syntactically Awesome Stylesheets” es como un css pero con muchas mas características que nos permiten usar variables, importar otras hojas de estilo, poner etiquetas dentro de otras, A primera vista no es muy diferente de un archivo CSS. Normalmente tiene la extensión .scss y también puede tener la extensión .sass pero es menos utilizado ejemplo:

En la imagen podemos ver el archivo.scss y a la derecha el archivo final .css Como podemos ver a la izquierda, el archivo .scss comienza con la definición de dos variables ($main y $enlaces) a las que le asignamos un color a cada una y que luego usamos más abajo. Luego a continuación nosfijamos cómo está anidado el código y lo que genera una vez compilado en CSS.

Como compilar

Primero hay que intalar ruby en nuestra pc, luego el siguiente comando,
gem install sass
sass hoja_estilos.scss hoja_estilos.css
