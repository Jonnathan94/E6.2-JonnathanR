Estructura de un elemento

Margin

El Margin es la separación entre el borde y la parte exterior de un elemento.

Border

El Border es una línea que se dibuja alrededor de un elemento.

Padding

El Padding es la separación entre el borde y el elemento.

Content

El Content es el contenedor que albergar un dibujo o texto.

Propiedades de CSS

width

El widht se utiliza para ajustar el ancho de un elemento, para ajusatar elemento podemos usar algunos valores como el % px

max-width

El max-width se usa para fijar un ancho maximo de un elemento.

height

Se usa para ajustar la altura de un elemento, podemos usar algunos valores como el % o px.

max-heigth

El max-height se usar para fijar una altura maxima de un elemento

display

El display se usa para decir si un elemento estara en bloques o sera en linea.

overflow

El overflow se usa para hacer visibles o invisibles los elementos, con el valor overflow: hidden, hacemos que un elemento sea invisibles, pero elemento sigue ocupando sitio en la página html y con el valor overflow: visible el elemento vulve estar visible, con el valor overflow: none, hacemos que el elemento se invisible y aparte no ocupe espacio.

z-index

El z-indez usa para decir en que nivel se mostrara un elemento.

line-height

El line-height se usa para decirle que altura dentra cada línea que forma el contenido del texto

box-sizing

Se usa para modificar las propiedades por defecto del css box model que calculan el alto y el ancho de los elementos.
